# LSTM Next Word Predictor

A Deep Learning project that predicts the next word in a sentence using an LSTM (Long Short-Term Memory) neural network built with TensorFlow and Keras.

## Features

- Text preprocessing and tokenization
- Sequence generation for training
- LSTM-based next word prediction
- Interactive text prediction
- Implemented in Google Colab

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Pandas

## Project Structure

```
LSTM-Next-Word-Predictor/
├── LSTM_Next_Word_Predictor.ipynb
└── README.md
```

## How to Run

1. Clone the repository.
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Install the required libraries.
4. Run all cells to train the model and generate predictions.
